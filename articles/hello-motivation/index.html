Hello Motivation

24th March 2020 - 2 mins to read

(Ironically it took me ages to even begin writing this, although my brain is admittedly and understandably somewhat distracted right now...)

Up to this point, I've been absolutely awful at writing about my personal projects. Whether it's space data processing, plotter code writing, or ZOIA format deciphering, I've shared almost exactly no details about how or why I'm doing any of it. Due to a somewhat miscalibrated belief that things I create need to be perfect before I send them out into the world, I err on the side of saying nothing whatsoever (with the main exception being Areo Browser, a site that I'm immensely proud of, but also deem finished enough to be a complete product that can be safely be presented as such).

The other reason is that it's always been too tricky to simply share things. I've had a few personal sites in the past, but none of them made it easy enough to share code, data, live ThreeJS/WebGL content, or tutorials in the way that I wanted.

Which brings me to this post on this new site. Written using Sapper and TailwindCSS, and hosted for free on Github Pages, it finally gives me a way to write the probably short, almost-certainly incredibly niche posts that have been swirling around in my brain for the last few years.

Going forwards, my goal is to write more often (it's a low bar), open source as much code as possible, and try to become more comfortable with not waiting until a project is finished before sharing it.
